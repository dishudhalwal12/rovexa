# Rovexa

Premium marketing site scaffold for Rovexa.

## Stack

- Next.js 16
- App Router
- TypeScript
- Tailwind CSS v4
- Framer Motion
- GSAP-ready animation layer
- Firebase client setup
- MDX-ready blog architecture
- Cloudflare deployment via OpenNext

## Commands

```bash
npm install
npm run dev
npm run build
npm run cf:build
npm run cf:preview
```
